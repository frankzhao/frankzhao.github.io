DFRobot I2C LCD1602 (Gravity)

Jan 12, 2019 in ELECTRONICS
electronics arduino
1 min read

This LCD is a small I2C display module that has a 16x2 character display. The text appears white on blue, with a dimmable backlight.

Note that this display uses a AiP31068L chipset, that is not compatible with the more common HD44780 chipset. Information and libraries for this chipset were a little hard to find, so I have listed some resources here for reference.

  • DFRobot wiki
  • Arduino library
  • Datasheet

The officially provided library appears to be some monochromatic re-implementation of Seeed Studio’s Grove LCD RGB Backlight that can be found at: https://github.com/Seeed-Studio/Grove_LCD_RGB_Backlight

I could not find a datasheet for this chipset, however I came across the information sheet which had some broken links. (http://www.west-l.com/uploads/tdpdf/104030001_eng_tds.pdf)
